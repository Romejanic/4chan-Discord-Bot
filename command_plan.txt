/4chan
	Gets bot info (eq +4chan info)

/4chan help
	Lists all commands and their usage (eq +4chan help)

/random ?board:[choice of boards]
	Gets a random post from either the default board or a
	specified board (eq +4chan random/+4chan <board>)

/post id:string board:[choice of boards]
	Gets the OP post of a thread with a given ID on a given
	board (eq +4chan post)

/config default_board set board:[choice of boards]
	Sets the new default board for the random command
	(eq +4chan config default_board <board>)

/config default_board reset
	Resets the default board back to the global default
	(eq +4chan config default_board clear)

/config removal_time set seconds:integer
	Sets the number of seconds the user has to react with
	an :x: to remove a post (eq +4chan config removal_time <seconds>)

/config removal_time reset
	Resets the removal time back to the global default
	(eq +4chan config removal_time clear)

/config allowed_channels toggle channel:channel
	Toggles whether a certain channel can be used by the
	bot (eq +4chan config allowed_channels #<channel>)

/config allowed_channels reset
	Resets the allowed channels list back to its default, which
	is any NSFW channel (eq +4chan config allowed_channels reset)

?option = optional argument
